Movie API

User can access a list of movies, get information about a specified movie or director. Users can also sign up for an account, create a profile and favorites list.

Resource URL

http://127.0.0.1:8080

Requests, URL and responses

Request Type Endpoint URL Query Parameter Body Format Description Response Format
GET '/movies' - - Get list of ALL movies JSON
GET '/movies/{movie_title}' movie_title - Get data about a specific movie title JSON
GET '/movies/genres/{genere_name}' genere_name - Get list of movies of a specific genere JSON
GET '/movies/director/{director_name}' director_name - Get data about a specific director JSON
POST '/users' - JSON
jsonNameString
Register a new user JSON
PUT '/users/{username}' account_id JSON
jsonNameStringUpdate
Let user update their information JSON
POST '/users/{user_name}/movies/{movie_id}' account_id, movie_name - Let user add movie to favorites Text
DELETE '/users/{user_name}/movies/{movie_id}' account_id, movie_name - Let user remove movie from favorites list Text
DELETE '/users/{user_name}' account_id - Allow user to remove their account Text
